建造者模式

---

**題目：**  
假設你在開發一個點餐系統，客戶可以根據喜好訂製**漢堡**。每個漢堡可以選擇以下內容：

1. **麵包種類**：全麥、白麵包
2. **肉類**：牛肉、雞肉、素肉
3. **配料**：生菜、番茄、洋蔥、醃黃瓜
4. **醬料**：番茄醬、美乃滋、芥末醬

請使用**建造者模式 (Builder Pattern)** 在 C# 中設計此系統，讓客戶能夠靈活地創建各式各樣的漢堡。  
建構的過程需要將麵包、肉類、配料和醬料逐步加入，並在最後「建造」完整的漢堡。每個漢堡的具體搭配應是可選的（即可以忽略某些部分，例如不加醬料或配料）。  

---

### **要求：**
1. 建立一個 `Burger` 類別，代表最終產品。
2. 使用建造者模式設計 `BurgerBuilder`，能夠靈活地設置漢堡的各部分。
3. 建立 `Director` 類別來演示常見的預設組合（例如「牛肉漢堡」、「素食漢堡」）。
4. 程式碼必須清晰簡潔，並能展示出如何使用此模式創建自訂漢堡。
